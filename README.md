# LASTD-2026: Linux Adaptive Scheduling and Threat Detection Dataset

## Overview

LASTD-2026 (Linux Adaptive Scheduling and Threat Detection Dataset) is a controlled Linux runtime dataset developed to support research on AI-driven operating systems, adaptive process scheduling, intelligent threat detection, and security-aware resource management.

The dataset was developed as part of the SecureSchedAI framework for evaluating Deep Reinforcement Learning-based scheduling and LSTM-based threat detection in Linux environments.

## Dataset Statistics

| Class                |    Samples |
| -------------------- | ---------: |
| Normal Activity      |     50,000 |
| Ransomware           |      5,000 |
| Brute Force          |      5,000 |
| Privilege Escalation |      5,000 |
| Suspicious Process   |      5,000 |
| **Total**            | **70,000** |

## Features

The dataset contains runtime Linux operating system metrics, including:

* CPU Utilization
* Memory Usage
* Process Queue Length
* I/O Waiting Time
* System Call Activity
* Process Behavior Indicators
* File Access Events
* Network Telemetry
* Threat Labels

## Applications

LASTD-2026 can be used for:

* Adaptive Process Scheduling
* Reinforcement Learning Research
* Linux Runtime Optimization
* Intrusion Detection Systems (IDS)
* Threat Detection and Classification
* AI-Assisted Operating Systems
* Cybersecurity Analytics
